# Pattern: Anti-slop UI

Ban or discourage:

- Inter/Roboto-only stacks, purple gradients, nested cards, gray-on-color text  
- Emoji-as-icons, bounce easing, empty “AI SaaS” hero grids  

Require:

- Explicit aesthetic direction, type pairing, spacing rhythm  
- Mobile + desktop hierarchy, real content not lorem  
- Skills/DESIGN.md when present
